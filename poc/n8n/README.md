# n8n workflow to process PDF reports

Directory structure:

- **n8n_data**: Contains data generated by n8n workflows, such as databases, logs, and other outputs.
- **n8n_io**: Contains the working directory for n8n. All input files, helper scripts and output files are stored here.
- **n8n_workflow**: Contains the n8n workflow files that automate the processing of PDF reports.

## quickstart

- create the .venv virtual environment in the `n8n_io` directory and install the required packages (python3 required on local machine)
- docker compose up -d
- open http://localhost:5678
- import the workflow from `n8n_workflow` directory
- coppy in n8n_io the config.json.example to config.json and edit it with your settings
- run the workflow in n8n (allPdfsToJson)
