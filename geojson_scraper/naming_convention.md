| **Key**             | **Description**                                              |
| ------------------- | ------------------------------------------------------------ |
| `biodata`           | Root key containing all venue-related data                   |
| `associated_names`  | All names that are associated with the venue                 |
| `place`             | Street address, city, and country if available               |
| `coordinates`       | Latitude and longitude as a comma-separated string           |
| `coordinates_trust` | Indicates the confidence level of the coordinates            |
| `year`              | Year of the Olympic event                                    |
| `season`            | Season of the Olympics (Summer, Winter)                      |
| `sports`            | Array of sports hosted at the venue                          |
| `opened`            | Year the venue was opened                                    |
| `seating_capacity`  | Number of spectators the venue can accommodate               |
| `surface`           | The type of surface of the venue (e.g. artifical, natural)   |
| `type`              | Description of the venue type (e.g., temporary indoor venue) |
